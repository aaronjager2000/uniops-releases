# uniops-releases

Public UniOps desktop download mirror.

## Latest macOS build

Version: `0.1.4`

Apple Silicon:

```bash
curl -L -o UniOps-mac-arm64.dmg https://raw.githubusercontent.com/aaronjager2000/uniops-releases/main/downloads/uniops-mac-arm64.dmg
```

Intel Mac:

```bash
curl -L -o UniOps-mac-x64.zip https://raw.githubusercontent.com/aaronjager2000/uniops-releases/main/downloads/uniops-mac-x64.zip
```

Checksums:

```bash
curl -L https://raw.githubusercontent.com/aaronjager2000/uniops-releases/main/downloads/SHA256SUMS
```

These artifacts are unsigned local builds until the UniOps code-signing
pipeline has production certificates.
